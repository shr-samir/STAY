STAY - Roomfinder Web Application

Project “STAY” is progressive web application designed to address the problem of room finding for students and families who have come with bigger dreams and for other different kinds of opportunities to make their life better.This web application is designed to connect tenant directly to houseowner. It helps tenants to save their time instead of roaming around for room hunting which may take days to get the desired room.
